# Sorting Algorithms : Empirical Asymptotic Analysis

Comparar a eficiência entre os algoritmos baseado em comparações

- SELECTION SORT
- INSERTION SORT
- MERGE SORT
- HEAP SORT
- QUICK SORT

## Requirements

- C Compiler (gcc)
- GNUPlot

## Usage

Run make command on root project folder

```
make
```

Run sorter.exe generated program, on Windows :

```
.\sorter.exe
```
